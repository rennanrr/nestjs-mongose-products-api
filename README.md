## Tech Stack
- Nest
- Typescript
- Mongoose
- Docker (MongoDB)

## Routes:
GET - /products
POST - /products
UPDATE - /products
DELETE - /products

## Get Started
First, make sure you have [NodeJS](https://nodejs.org/en/download/).
## Docker

There is a `docker-compose.yml` file for starting MongoDB with Docker.

`$ docker-compose up`

After running, you can stop the Docker container with

`$ docker-compose down`

### Install packages
```
npm install
```

### Run Server in development mode
```
npm run start:dev
```

### Run Server
```
npm run start
```
